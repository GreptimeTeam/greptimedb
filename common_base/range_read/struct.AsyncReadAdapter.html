Expand description

AsyncReadAdapter adapts a RangeReader to an AsyncRead.

Fields§

§inner: Arc<Mutex<R>>

The inner RangeReader. Use Mutex to get rid of the borrow checker issue.

§position: u64

The current position from the view of the reader.

§buffer: Vec<u8>

The buffer for the read bytes.

§content_length: u64

The length of the content.

§read_fut: Option<Pin<Box<dyn Future<Output = Result<Bytes>> + Send>>>

The future for reading the next bytes.

Struct Plugins

pub struct Plugins {
    inner: Arc<RwLock<SendSyncAnyMap>>,
}
Expand description

Plugins is a wrapper of anymap2 and provides a thread-safe way to store and retrieve plugins. Make it Cloneable and we can treat it like an Arc struct.

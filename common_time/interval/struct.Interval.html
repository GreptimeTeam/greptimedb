Struct common_time::interval::Interval

pub struct Interval {
    months: i32,
    days: i32,
    nsecs: i64,
    unit: IntervalUnit,
}
Expand description

Interval Type represents a period of time. It is composed of months, days and nanoseconds. 3 kinds of interval are supported: year-month, day-time and month-day-nano, which will be stored in the following format. Interval data format: | months | days | nsecs | | 4bytes | 4bytes | 8bytes |

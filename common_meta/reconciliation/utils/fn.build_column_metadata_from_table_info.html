Expand description

Constructs a vector of [ColumnMetadata] from the provided table information.

This function maps each [ColumnSchema] to its corresponding [ColumnMetadata] by determining the semantic type (Tag, Timestamp, or Field) and retrieving the column ID from the name_to_ids mapping.

Returns an error if any column name is missing in the mapping.
